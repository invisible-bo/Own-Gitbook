# VI(visual editor)

### Vi 기본 사용법

1. vi 시작하기

*   파일을 열기:

    ```bash
    vi <파일 이름>
    ```
*   새 파일 생성:

    ```bash
    vi <새 파일 이름>
    ```

2. **모드 전환**
   * **명령 모드** (기본 모드): 아무 키도 누르지 않은 초기 상태.
   * **입력 모드로 전환**:
     * `i`: 커서 앞에 텍스트 입력
     * `a`: 커서 뒤에 텍스트 입력
   * **명령 모드로 돌아가기**:
     * `Esc` 키
3. **명령 줄 모드**
   * **저장 및 종료**:
     * 저장하고 종료: `:wq`
     * 저장하지 않고 종료: `:q!`
     * 저장만: `:w`
