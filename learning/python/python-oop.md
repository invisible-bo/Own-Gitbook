# Python OOP

* Python의 객체 지향 프로그래밍(Object-Oriented Programming, OOP)은 \
  Data를 객체로 모델링하고, 이 객체들 간의 상호작용을 중심으로 프로그램을 설계하는 \
  Programming 패러다임. Python은 객체 지향 언어로, 모든 것이 객체로 구성되어 있다

## 객체지향프로그래밍의 주요 개념

### 1. Class
