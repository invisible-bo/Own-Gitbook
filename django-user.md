# Django User

Django의 기본 `User` 모델의 기본 필드와 기본 기능

### 1. 기본 필드

* username
* password
* email
* first\_name
* last\_name
* 기타 인증과 관련된 필드 (`is_staff`, `is_active`, `is_superuser`, `last_login`, `date_joined` 등)

### 2. 기본 기능

* 사용자 인증 (로그인, 로그아웃)
* 관리자 페이지에서 사용자 관리
* 권한 및 그룹 관리
